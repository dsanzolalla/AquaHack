# **App Name**: AquaHack

## Core Features:

- Issue Reporting via Chatbot: Users can report water contamination issues via a chatbot that supports voice and text in multiple local languages. The LLM uses a tool to determine the issue type, severity and location, and provides immediate guidance.
- Automated Report Generation: The chatbot automatically generates a detailed report from the user's input and sends it to relevant authorities.
- Interactive Contamination Map: Display an interactive map showing reported contamination incidents, their status, and relevant details, filterable by issue type, severity, and date.
- Offline Accessibility: Enable core features to be accessible offline, with optimized data usage for basic mobile devices.
- Educational Resources: Provide educational content on water contamination risks, symptoms, and preventive measures.
- Alert Notifications: Send notifications about nearby contamination incidents, recommendations, or relevant updates from authorities.
- Authorities Reporting Panel: A dashboard for authorities to view reports, update statuses, and provide feedback.

## Style Guidelines:

- Primary color: Deep blue (#1A535C) evoking water and safety.
- Background color: Light blue (#E6F0F2) for a clean and fresh feel.
- Accent color: Orange (#FF6B6B) to highlight critical alerts and calls to action, indicating urgency.
- Body and headline font: 'PT Sans' (sans-serif) for clear readability across different languages.
- Use simple, culturally relevant icons to represent different types of water contamination and actions.
- Design a clean and intuitive interface optimized for low data usage and readability on basic mobile devices.
- Use subtle animations to provide feedback on user actions, such as submitting a report or updating the map.
