💧 AquaHack
Proyecto desarrollado en la V Edición #Hack4Edu por la Universidad Pontificia de Salamanca, con el apoyo de Profuturo.

📖 Descripción del Proyecto
AquaHack es una aplicación web interactiva diseñada para abordar desafíos relacionados con la gestión y el acceso al agua. Su propósito principal es empoderar a las comunidades brindándoles herramientas intuitivas para monitorizar recursos hídricos, reportar incidencias y educarse al respecto.

✨ Características Principales
La plataforma cuenta con 4 módulos principales accesibles desde el panel de control:

🗺️ Mapa Interactivo: Permite la visualización geográfica de puntos críticos, estado de los recursos hídricos y reportes ciudadanos en tiempo real utilizando Google Maps.
🚰 Agua Potable: Información, monitorización y seguimiento sobre el acceso a fuentes de agua potable seguras.
📚 Recursos: Repositorio de material educativo, guías y consejos relacionados con el uso eficiente, cuidado del agua y sostenibilidad ambiental.
🛡️ Autoridades y Reportes: Un canal directo y estructurado para reportar incidencias, problemas de infraestructura, cortes en el suministro o alertas de calidad del agua a las autoridades responsables.

🛠️ Tecnologías Integradas
El proyecto está construido bajo una arquitectura moderna web con tecnologías clave:

Frontend: Next.js 15 (App Router), React 18, TypeScript.
Estilos y Componentes: Tailwind CSS sumado a componentes pre-diseñados y accesibles de Radix UI.
Mapas: Integración con Google Maps vía @vis.gl/react-google-maps.
Manejo de Formularios: react-hook-form y validaciones con zod.
Backend / Hosting: Preparado para el ecosistema de Firebase (apphosting).
Internacionalización: Implementación de traducciones nativas (multi-idioma).
